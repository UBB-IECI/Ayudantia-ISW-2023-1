# Codigos de las clases de ayudantia

----
## Instalacion en local

- Clonar el repositorio o copiar el codigo
- Posicionarse en la carpeta de la clase
- Instalar dependencias
    - `npm install`
    - `npm install -g nodemon`
- Ejecutar el servidor 
    - `npm run dev`
